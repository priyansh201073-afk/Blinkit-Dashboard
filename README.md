🛒 Blinkit Sales & Outlet Performance Dashboard — Power BI

An interactive Power BI dashboard analyzing Blinkit (India's Last Minute App) sales performance, outlet distribution, and product insights.


📊 Project Overview
This project presents a comprehensive Business Intelligence dashboard built in Power BI for Blinkit, India's quick-commerce grocery delivery platform. The dashboard provides actionable insights into sales trends, outlet performance, item categories, and customer ratings across multiple dimensions.

🔍 Key Metrics Tracked
MetricValue💰 Total Sales1.20M📦 No. of Items8,523💵 Avg Sales$141⭐ Avg Rating3.9

📌 Dashboard Highlights

Fat Content Analysis — Sales breakdown by Low Fat vs Regular products across outlet tiers
Item Type Performance — Revenue comparison across 15+ categories (Fruits, Snacks, Dairy, Frozen Foods, etc.)
Outlet Type Breakdown — Performance across Grocery Stores, Supermarket Type 1/2/3
Outlet Size Distribution — Sales split across Small, Medium, and High outlet sizes
Outlet Location Tiers — Tier 1, Tier 2, Tier 3 city-level performance
Outlet Establishment Trend — Year-wise sales growth from 2012 to 2022
Interactive Filters — Slicers for Outlet Location Type, Outlet Size, and Item Type
